# Sales Data Analysis Project

## Overview
This project analyzes a retail sales dataset to generate business insights using Python and MySQL.

## Tools Used
- Python
- Pandas
- Matplotlib
- MySQL
- SQL

## Workflow
1. Loaded CSV dataset using Pandas
2. Performed data cleaning and preprocessing
3. Created features for time-based analysis
4. Visualized sales patterns
5. Exported cleaned data to MySQL
6. Performed SQL business analysis queries

## Business Questions Answered
- What is the total revenue generated?
- Which product category generates the most sales?
- Which region has the highest revenue?
- Who are the top customers?
- What are the top selling products?
- How does revenue change by month?

## Advanced SQL Analysis
- Product revenue ranking using window functions
- Running total of monthly sales
